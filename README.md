# IOT-Inverter

### Build executable file

1. mkdir build
2. cd build
3. Release : cmake -DCMAKE_BUILD_TYPE=Release ..
   Debug :   cmake -DCMAKE_BUILD_TYPE=Debug ..

### Execution

1. cd build
2. cp ../docs/solar.conf  .
3. ./invIOT

### Trace Mode

1. cd build
2. cp ../docs/solar.conf  .
3. ./invIOT -T
